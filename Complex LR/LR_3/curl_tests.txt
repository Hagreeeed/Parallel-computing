=============================================================
     MARKETPLACE API - ПОВНИЙ ДОВІДНИК CURL ЗАПИТІВ
=============================================================

ЗЦей файл містить готові команди для тестування API напряму з терміналу.
Змініть EMAIL, PASSWORD, TOKEN та ID на необхідні.

> УВАГА: Користувач-адміністратор створюється автоматично при запуску сервера!
> Емейл: admin@marketplace.com
> Пароль: password значення.

=============================================================
 1. AUTH (АВТОРИЗАЦІЯ)
=============================================================
# Реєстрація (CUSTOMER):
curl -X POST http://localhost:8080/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{"email":"user@test.com", "password":"password", "firstName":"Test", "lastName":"User"}'

# Логін (отримання token):
curl -X POST http://localhost:8080/api/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"user@test.com", "password":"password"}'

# Оновлення токена (Refresh):
curl -X POST http://localhost:8080/api/auth/refresh \
  -H "Content-Type: application/json" \
  -d '{"refreshToken":"YOUR_REFRESH_TOKEN"}'

# Логаут:
curl -X POST http://localhost:8080/api/auth/logout \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

=============================================================
 2. USERS (КОРИСТУВАЧІ)
=============================================================
# Всі користувачі (GET):
curl -X GET http://localhost:8080/api/users -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Користувач за ID (GET):
curl -X GET http://localhost:8080/api/users/{id} -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Оновити користувача (PUT):
curl -X PUT http://localhost:8080/api/users/{id} \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"email":"new@test.com", "password":"new", "role":"CUSTOMER"}'

# Видалити користувача (DELETE):
curl -X DELETE http://localhost:8080/api/users/{id} -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Стати продавцем:
curl -X POST http://localhost:8080/api/users/{id}/become-seller \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"shopName":"My Shop", "description":"Shop Desc"}'

 8. ORDERS (ЗАМОВЛЕННЯ)
=============================================================
# Всі замовлення (GET):
curl -X GET http://localhost:8080/api/orders -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Мої замовлення (GET):
curl -X GET http://localhost:8080/api/orders/my -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Замовлення за ID (GET):
curl -X GET http://localhost:8080/api/orders/{id} -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Створити замовлення (POST):
curl -X POST http://localhost:8080/api/orders \
  -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" \
  -d '{"customerId":1, "items":[{"productId":1, "quantity":1, "priceAtTime":1500.0}]}'

# Змінити статуси (POST via custom paths):
curl -X POST http://localhost:8080/api/orders/{id}/confirm -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
curl -X POST http://localhost:8080/api/orders/{id}/ship -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
curl -X POST http://localhost:8080/api/orders/{id}/deliver -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
curl -X POST http://localhost:8080/api/orders/{id}/cancel -H "Authorization: Bearer YOUR_ACCESS_TOKEN"

# Оновити цілком (PUT):
curl -X PUT http://localhost:8080/api/orders/{id} -H "Authorization: Bearer YOUR_ACCESS_TOKEN" \
  -H "Content-Type: application/json" -d '{"totalAmount": 1500.0, "status": "PENDING"}'

# Видалити (DELETE):
curl -X DELETE http://localhost:8080/api/orders/{id} -H "Authorization: Bearer YOUR_ACCESS_TOKEN"
